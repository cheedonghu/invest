# 股票数据解析展示工程

这是一个最小可运行的股票估值展示项目：

- 后端使用 Python + FastAPI
- 数据源使用 AKShare
- 前端使用原生 HTML/CSS/JavaScript + ECharts
- 示例股票为招商银行（`600036`）
- 前端会基于历史 PE 序列拟合正态分布，并计算 20%、50%、80% 三条辅助线

## 启动方式

1. 安装依赖：

```bash
pip install -r requirements.txt
```

2. 启动服务：

```bash
uvicorn backend.app.main:app --reload
```

3. 打开浏览器访问：

```text
http://127.0.0.1:8000
```
